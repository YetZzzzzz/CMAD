You can train the **teacher model** from scratch using the following code:
```
python3 train_teacher_model.py --dataset='mosei' --d_l=96 --depth=5 --latent_dim=96 --learning_rate=2e-5 --n_epochs=80 --te_layers=2 --train_batch_size=128
```

Or you can also try to download the [checkpoint](https://pan.baidu.com/s/1bioKGv393xl7JjYXHGf8Ow?pwd=px6s) (Extraction code: px6s) we provided here.
